# App de bandeja (Fase 2) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Trocar o "invólucro" do app Electron de uma janela normal (1000x700, na barra de tarefas) para um ícone de bandeja + popup sem bordas, sem tocar no conteúdo (abas, IPC, watcher).

**Architecture:** `app/src/main/popupWindow.ts` cria a janela popup (frameless, escondida por padrão, esconde ao perder foco). `app/src/main/tray.ts` cria o ícone da bandeja, o menu de "Sair" e a lógica de mostrar/esconder/posicionar o popup. `app/src/main/index.ts` orquestra os dois e perde a lógica antiga de janela normal + `window-all-closed`.

**Tech Stack:** Electron (`Tray`, `Menu`, `BrowserWindow`), TypeScript — sem novas dependências.

## Global Constraints

- Nenhuma mudança no renderer (React, abas, IPC) — só no processo principal.
- Sem testes automatizados novos (infraestrutura de UI nativa, sem lógica de negócio testável).
- Windows-first — comportamento em outros SOs não é garantido nesta fase.
- Identificadores de código em inglês; texto voltado ao usuário final em português (o item de menu "Sair").

### Task 1: `app/src/main/popupWindow.ts` — janela popup sem bordas

**Files:**
- Create: `app/src/main/popupWindow.ts`

**Interfaces:**
- Consumes: nada de outras tasks.
- Produces: `createPopupWindow(): BrowserWindow`, `POPUP_WIDTH: number`, `POPUP_HEIGHT: number` — usados por `app/src/main/tray.ts` (Task 2) e `app/src/main/index.ts` (Task 3).

- [ ] **Step 1: Criar `app/src/main/popupWindow.ts`**

```ts
import { BrowserWindow } from 'electron';
import { join } from 'node:path';

export const POPUP_WIDTH = 380;
export const POPUP_HEIGHT = 500;

export function createPopupWindow(): BrowserWindow {
  const popupWindow = new BrowserWindow({
    width: POPUP_WIDTH,
    height: POPUP_HEIGHT,
    show: false,
    frame: false,
    resizable: false,
    skipTaskbar: true,
    webPreferences: {
      preload: join(__dirname, '../preload/index.js'),
      contextIsolation: true,
      nodeIntegration: false,
    },
  });

  if (process.env['ELECTRON_RENDERER_URL']) {
    popupWindow.loadURL(process.env['ELECTRON_RENDERER_URL']);
  } else {
    popupWindow.loadFile(join(__dirname, '../renderer/index.html'));
  }

  popupWindow.on('blur', () => {
    popupWindow.hide();
  });

  return popupWindow;
}
```

Isso é uma extração/adaptação de `createWindow()` de `app/src/main/index.ts` (que ainda existe até a Task 3 remover): mesmo `webPreferences`/`loadURL`/`loadFile`, mas `show: false` (começa escondida), `frame: false` e `skipTaskbar: true` (sem bordas, sem ícone na barra de tarefas), tamanho fixo 380x500 em vez de 1000x700, e um handler de `blur` que esconde a janela em vez de deixá-la aberta.

### Task 2: `app/src/main/tray.ts` — ícone da bandeja, menu e posicionamento

**Files:**
- Create: `app/src/main/tray.ts`

**Interfaces:**
- Consumes: `POPUP_WIDTH`, `POPUP_HEIGHT` de `./popupWindow` (Task 1).
- Produces: `createTray(popupWindow: BrowserWindow): Tray` — usado por `app/src/main/index.ts` (Task 3).

- [ ] **Step 1: Criar `app/src/main/tray.ts`**

```ts
import { app, BrowserWindow, Menu, Tray } from 'electron';
import { join } from 'node:path';
import { POPUP_HEIGHT, POPUP_WIDTH } from './popupWindow';

function positionPopup(popupWindow: BrowserWindow, tray: Tray): void {
  const trayBounds = tray.getBounds();
  const x = Math.round(trayBounds.x + trayBounds.width / 2 - POPUP_WIDTH / 2);
  const y = Math.round(trayBounds.y - POPUP_HEIGHT);
  popupWindow.setPosition(x, y, false);
}

function togglePopup(popupWindow: BrowserWindow, tray: Tray): void {
  if (popupWindow.isVisible()) {
    popupWindow.hide();
    return;
  }

  positionPopup(popupWindow, tray);
  popupWindow.show();
  popupWindow.focus();
}

export function createTray(popupWindow: BrowserWindow): Tray {
  const tray = new Tray(join(__dirname, '../../resources/icon.png'));
  tray.setToolTip('Prismly');

  tray.on('click', () => togglePopup(popupWindow, tray));

  const contextMenu = Menu.buildFromTemplate([{ label: 'Sair', click: () => app.quit() }]);
  tray.setContextMenu(contextMenu);

  return tray;
}
```

Nota sobre o caminho do ícone: `join(__dirname, '../../resources/icon.png')` assume que, no build de desenvolvimento (`electron-vite dev`), `__dirname` do processo principal compilado aponta pra `app/out/main/`, então `../../resources/icon.png` resolve pra `app/resources/icon.png` (o ícone que já existe no projeto, usado hoje só pra empacotamento). Isso só foi testado em modo dev — empacotamento/distribuição fica pra uma fase futura (fora de escopo, conforme a spec).

No Windows, `tray.setContextMenu(...)` já mostra o menu automaticamente no clique direito — não precisa de um handler `right-click` separado. O evento `'click'` (clique esquerdo) é tratado à parte, por isso `togglePopup` só é chamado ali.

### Task 3: Atualizar `app/src/main/index.ts` para usar o Tray + popup

**Files:**
- Modify: `app/src/main/index.ts`

**Interfaces:**
- Consumes: `createPopupWindow` de `./popupWindow` (Task 1), `createTray` de `./tray` (Task 2).
- Produces: nada consumido por outra task — é o ponto de entrada do processo principal.

- [ ] **Step 1: Substituir `app/src/main/index.ts`**

```ts
import { app, BrowserWindow, Tray } from 'electron';
import { collectClaudeUsage } from '../../../core/adapters/claude';
import { calculateCost } from '../../../core/pricing';
import { aggregateUsage } from '../../../core/aggregator';
import { computeBlocks } from '../../../core/blocks';
import { startWatcher } from './watcher';
import { createPopupWindow } from './popupWindow';
import { createTray } from './tray';
import type { UsagePayload } from '../shared/types';

function buildPayload(): UsagePayload {
  const records = collectClaudeUsage().map((record) => ({
    ...record,
    cost: calculateCost(record),
  }));

  return {
    aggregated: aggregateUsage(records),
    blocks: computeBlocks(records),
  };
}

let popupWindow: BrowserWindow | null = null;
let tray: Tray | null = null;

function sendUpdate(): void {
  if (!popupWindow) return;
  popupWindow.webContents.send('usage:update', buildPayload());
}

app.whenReady().then(() => {
  popupWindow = createPopupWindow();
  popupWindow.webContents.on('did-finish-load', sendUpdate);
  tray = createTray(popupWindow);
  startWatcher(sendUpdate);
});
```

Mudanças em relação à versão anterior:
- `createWindow()` (janela normal 1000x700) dá lugar a `createPopupWindow()` (Task 1) + `createTray()` (Task 2).
- O handler `app.on('window-all-closed', () => { if (process.platform !== 'darwin') app.quit(); })` foi removido — a janela nunca é "fechada" pelo usuário agora (só escondida via `blur`), e sair do app é sempre pelo menu "Sair" do clique direito (que chama `app.quit()` diretamente em `tray.ts`).
- `tray` é mantido numa variável de módulo (`let tray: Tray | null = null`) mesmo sem ser lido depois — isso evita um problema conhecido do Electron onde o ícone da bandeja pode desaparecer se a única referência ao `Tray` for coletada pelo garbage collector.
